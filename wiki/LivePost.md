#LivePost

消息实时推送

##接口调用请求说明

接口URL：http://apihost/news

请求方式：POST

参数说明：

|参数|类型|是否必须|范围|说明|
|:--|:--|:--|:--|:--|
|userid|字符串|必须|-|用户id|

##返回说明
|参数|类型|说明|
|:--|:--|:--|
|code|整型|操作码，1表示用户在线消息提醒成功，0用户不在线|

##示例

向用户实时推送消息提醒

http://apihost/news

    JSON:
    {
        "ret": 200,
        "data": {
            "code": 0,
            "msg": "该用户不在线",
        },
        "msg": ""
    }
